# 태양광 프로잭트

### 1. 개요 

2020/ 06~ 2020/ 09 

2020 빅데이터 청년인재 일자리 연계 사업에서 진행한 프로잭트.

### 2.  사용 데이터및 프로잭트 설명

1. 서울시 건물 데이터(높이,  좌표)
2. 날짜, 시간별 태양의 위치 데이터(고도, 방위각)
3. 태양광 패널의 각도에 따른 효율 데이터



### 3. 건물 벽면에 태양광 패널을 부착시 예상되는 발전량 계산 

1. 원하는 건물의 좌표 설정
2.  해당 건물 주위의 건물 데이터를 가져옴
3. 건물간의 거리와 태양의 위치를 이용한 선택 건물의 벽면 일조량 계산( 높이별 일조량)
4. 일조량과 시간별 태양의 고도에 따른 발전 패널의 효율 계산
5. 하루평균 발전량 도출
